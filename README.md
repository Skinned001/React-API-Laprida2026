# React-API-Laprida2026
Astillero Imperial: API de Gestión de Flota (Stellaris)
Construido con **Node.js, Express y Sequelize.

Antes de iniciar, asegúrate de tener instalados los siguientes sistemas en tu terminal local:

* [Node.js](https://nodejs.org/) (v16.0 o superior)
* Un servidor **MySQL** activo (XAMPP, WAMP, o MySQL Workbench)
* Puerto `3000` disponible en tu red local.

---
Instalación

1. **Clonar/Abrir el repositorio** en tu máquina local.
2. **Abrir la terminal** en la carpeta raíz del proyecto backend.
3. **Instalar los módulos de dependencia** ejecutando el siguiente comando:

npm install

# Configuración del Puerto del Servidor
PORT=3000

# Para ejecutar el servidor usa:
npm run dev
